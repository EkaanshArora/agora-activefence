Agora x ActiveFence

This project showcases the content inspect moderation features available with Agora and their partner ActiveFence.

To see the demo in action we recommend setting up a Kicking Server:
1. Head to agora-activefence-kicker and use the one-click deployment for the server.
2. Add the deployment link + /kick/ to the webhook URL for the desired event in the ActiveFence dashboard.
